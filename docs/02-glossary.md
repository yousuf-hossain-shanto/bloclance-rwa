# Glossary

| Term | Definition |
|---|---|
| **RWA** | Real-World Asset. Off-chain asset (real estate) represented on-chain as tokens. |
| **Unit** | Fractional share of a property. Each property has total units (e.g. 1,200) and price per unit (e.g. $430). |
| **ROI** | Return on Investment % shown per asset. Likely annualized rental yield. |
| **Hold period** | Recommended ownership horizon, e.g. "3–5 Years". |
| **XRP** | Native XRPL token. Used as one of two settlement assets. |
| **RLUSD** | Ripple USD stablecoin. Used as the other settlement asset. |
| **XRPL** | XRP Ledger. Settlement layer for the app (implied by branding "SurgeXRP" + wallet addresses starting with `r...`). |
| **Wallet** | User's on-chain account. App shows two: XRP Wallet and RLUSD Wallet (same `r...` address shown in mocks). |
| **Market Order** | Buy/sell at current market price. Fills immediately. |
| **Limit Order** | Buy/sell at user-specified price. Sits on the book until matched. |
| **Order Book** | Live list of buy/sell offers for an asset, with price and unit counts. |
| **Trade volume** | Total notional traded for an asset, e.g. *$12,587,971*. |
| **Property value / valuation** | Underlying real estate appraisal, e.g. *$2,581,023*. |
| **Total units sold / Available units** | Issuance progress, e.g. *428,587 sold / 71,413 available / 500,000 total*. |
| **Earnings** | Yield accrued from holdings, withdrawable via Withdraw flow. |
| **Human Verification** | KYC step. States: *Not Verified* / *Verified*. |
| **Surge Wallet** | App's branded wallets — XRP Wallet + RLUSD Wallet. |
